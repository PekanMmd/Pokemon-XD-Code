In order to use these tools first double click on 'installer.bat'
  - This may take a few minutes and you may be prompted to give permission to the installers.
  - You only need to do this once. Once everything is set up you can use the tool like normal.

To use the tool simply drag and drop the file you want to open unto the .exe file and a window will appear.
  - The tool supports .iso, .fsys, .msg, .gtx and .atx files.
